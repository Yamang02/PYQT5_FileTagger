- **요청**: 프로젝트 기획 시작 제안.
- **논의**: 
  - **핵심 목표**: 기존 파일들에 태그를 부여하여, 여러 디렉토리에 흩어진 파일들에 대해 태그를 기반한 파일 브라우징 및 관리를 효율적으로 수행.
  - **해결 문제**: 여러 폴더 탐색 비효율성 해소, 파일 분류 어려움 해결, 비슷한 성격 파일 접근성 향상.
  - **핵심 방향**: 수동 태깅에 최적화된 구성.
  - **성공 기준**: 1차 목표로 '태깅 작업 효율성' (파일 하나에 태그 추가 평균 3초 이내) 설정.
